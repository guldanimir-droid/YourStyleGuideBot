# Wardrobe Bot

Telegram-бот для управления гардеробом. Позволяет добавлять вещи, просматривать гардероб, удалять и получать случайные образы.

## Команды
- /start — главное меню
- /add_clothes — добавить вещь
- /my_wardrobe — показать гардероб
- /look — что надеть сегодня
- /delete_clothes — удалить вещь (через кнопки)

## Технологии
- Python, aiogram, Supabase
